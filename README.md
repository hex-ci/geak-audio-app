# Geak Audio 果壳智能音响桌面工具

支持 Windows、Mac 和 Linux

## 功能

* 支持云听电台
* 支持喜马拉雅电台
* 支持喜马拉雅频道
* 支持本地播放列表
* 支持收藏喜爱的电台或频道

## 开发

请先安装 node js，推荐 14.x

然后执行 `npm i`

## 调试运行

`npm run electron:serve`

### 构建当前系统的可执行文件

`npm run electron:build`
